# MiniYT-Backend

A backend implementation of a video-sharing platform, focused on mastering REST API architecture, database management, and server-side logic.

## 🚀 Project Overview
MiniYT-Backend is a project built to explore the fundamentals of backend engineering. It handles the core logic required for a video platform, including data persistence and API communication.

## 🛠️ Tech Stack
* **Language:** Python
* **Database:** SQL
* **Environment:** Linux (Ubuntu)
* **Architecture:** RESTful API

## ⚙️ Core Features
* **REST API Endpoints:** Implementation of standard HTTP methods (GET, POST, PUT, DELETE) for resource management.
* **Database Integration:** Structured data storage using SQL for managing user and video metadata.
* **Linux Development:** Developed and tested within a VirtualBox Linux environment to ensure system-level compatibility.
